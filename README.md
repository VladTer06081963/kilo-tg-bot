# 🤖 Kilo CLI Telegram Bot

Telegram-бот для управления Kilo CLI проектами. Позволяет создавать проекты, выполнять команды и управлять файловой структурой через Telegram.

## 📋 Возможности

- 📦 **Создание проектов** — автоматическое создание Express + Pug проектов
- ▶️ **Выполнение команд** — запуск произвольных команд через Kilo CLI
- 📂 **Управление проектами** — просмотр, создание и удаление проектов
- 🔄 **Real-time вывод** — получение результатов выполнения команд в реальном времени

## 🚀 Установка

```bash
cd telegram-kilo-bot
npm install
```

## ⚙️ Настройка

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Скопируйте токен бота
3. Отредактируйте файл `.env`:

```env
TELEGRAM_TOKEN=ВАШ_ТОКЕН_БОТА
```

## 🎮 Команды бота

| Команда | Описание |
|---------|----------|

| `/start` | Запуск бота |
| `/help` | Справка |
| `/create [имя]` | Создать Express + Pug проект |
| `/list` | Список проектов |
| `/delete [имя]` | Удалить проект |
| `/run [команда]` | Выполнить команду через Kilo |
| `/status` | Статус Kilo CLI |

## 📁 Структура проекта

```text
telegram-kilo-bot/
├── bot.js           # Основной файл бота
├── .env             # Переменные окружения
├── package.json     # Зависимости
└── projects/        # Созданные проекты
    └── [имя проекта]/
```

## 🛠️ Примеры использования

### Создание проекта

```bash
/create myapp
```

### Выполнение команды

```bash
/run Добавь аутентификацию
```

### Удаление проекта

```bash
/delete myapp
```

## 🔧 Требования

- Node.js 18+
- Kilo CLI 7.0+
- Telegram бот
